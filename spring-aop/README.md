使用DI管理和配置应用对象，DI有助于应用对象之间的解耦。而AOP可以实现横切关注点和他们所影响的对象之间的解耦。

## 术语：

### 通知：

切面的工作，切面所需要完成的事情。通知定义了切面是什么，以及如何使用

#### 前置通知

目标方法被调用之前调用通知功能

#### 后置通知

目标方法被调用之后调用通知功能

#### 返回通知

在目标方法成功执行之后调用通知

#### 异常通知

在目标方法抛出异常调用通知

#### 环绕通知

通知包裹被通知方法，在被通知方法调用前后执行

### 切点：

切入何处



#### 切面：

切点和通知的合体
